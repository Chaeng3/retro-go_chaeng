## ✨ Features

- 基于 Retro-Go (ducalex) 二次开发
- ESP32-S3 双核高性能平台
- 3.2 寸 TFT LCD（SPI 320×240）
- SD 卡游戏加载系统
- I2S 音频输出（喇叭驱动）
- 完整按键控制系统
- WS2812 RGB 状态灯（GPIO38）
- 全中文代码与注释
- 可扩展 WiFi / AI / 联机功能

---

## 🌈 RGB Status LED

- Type: WS2812
- GPIO: 38
- Default: Blue

States:
- 🟦 Boot
- 🟩 Running
- 🟥 Error
